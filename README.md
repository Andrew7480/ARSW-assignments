# ARSW Assignments

This repository contains the assignments, workshops, and practical exercises developed throughout the **Software Architecture (ARSW)** course.

## About the Course

Software Architecture focuses on the design and construction of software systems that are scalable, maintainable, reliable, and efficient. Throughout the course, different architectural concepts, design principles, and implementation techniques are explored through hands-on assignments and practical exercises.

Some of the topics covered include:

- Concurrency and multithreading
- Synchronization mechanisms
- Design patterns
- Software quality attributes
- Distributed systems concepts
- Architectural styles and principles
- Performance and scalability considerations

## Repository Structure

The repository is organized by assignments and topics. Each folder contains the source code, documentation, and any additional resources required for a specific exercise.

```text
ARSW-assignments/
├── introduction-to-threads/
├── synchronization/
├── producer-consumer/
├── design-patterns/
└── ...
```

Each directory may include:

- Source code
- Instructions and requirements
- Documentation
- Test cases
- Additional resources

## Purpose

The goal of this repository is to keep track of the work completed during the course and serve as a reference for the concepts and techniques learned throughout the semester.
